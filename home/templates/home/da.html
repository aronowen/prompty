{% blocktrans trimmed %} You are a Data Analyst at a growing SaaS company that provides a web platform for small businesses. Each month you produce a Product Usage and Adoption Dashboard that tracks how actively customers are engaging with key features. {% endblocktrans %}

{% blocktrans trimmed %} This month’s dashboard highlights several notable patterns. Overall logins have increased, but one of the core product features has experienced a clear and meaningful drop in usage. New customers appear to be adopting key features more quickly than long-term customers, and churn risk is rising within a particular customer segment. Additionally, a recent UX change seems to have altered behaviour across two major user flows. {% endblocktrans %}

{% blocktrans trimmed %} Your task is to send a summary update to the leadership team and product managers. In this update you should explain what trends you observe in the data, why these trends matter for the business, and what potential decisions, risks, or follow-up questions emerge from the patterns you have identified. {% endblocktrans %}

{% blocktrans trimmed %} Write a prompt that you would give to an AI assistant to help you draft this summary. Your prompt should include clear context about the dashboard, the intended audience of non-technical leaders, and specific instructions about the structure of the summary, such as highlighting key insights, explaining the impact, and suggesting recommended next steps. {% endblocktrans %}

{% blocktrans trimmed %} Hint: Imagine you are briefing a consultant who has never seen your product or dataset before. What information would they need to produce a clear, confident, and actionable summary? {% endblocktrans %}
